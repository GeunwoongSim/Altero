## 🏗 How to Use 
### Require
- iOS 17+
### Build from source code
1. Clone/Download the repo
2. Open `Altero.xcodeproj` in Xcode
3. Choose the right target (iOS)
4. Build & run!
### 설치 및 실행
```bash
# 레포지토리 클론
git clone [https://github.com/Syrbear/Altero.git](https://github.com/Syrbear/Altero.git)

# 프로젝트 폴더로 이동
cd Altero

# Xcode 프로젝트 실행
open Altero.xcodeproj (또는 .xcworkspace)

```

📂 폴더 구조
```
Altero/
├── Sources/
│   ├── Models/         # 데이터 구조
│   ├── Views/          # UI 화면 (SwiftUI)
│   ├── ViewModels/     # 비즈니스 로직
│   └── Resources/      # Assets, Fonts 등
└── Tests/              # 유닛 테스트
```
